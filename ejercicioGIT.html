Examen Git y Github

Alumn@: Escribe tus iniciales: imc, he añadido nueva info

  1. ¿Que es un sistema de control de versiones?

    Herramienta que registra los cambios en archivos para recuperar versiones específicas y permitir el trabajo en equipo sin sobrescribir código.

  2. ¿Cual es la diferencia entre git y github?

    Git es el software local para gestionar el historial; GitHub es la plataforma en la nube para alojar y compartir esos repositorios.

  3. ¿Cómo creo una rama en git?

    git remote -v: Muestra las URLs de los servidores remotos conectados a tu proyecto local.

  4. ¿Cómo clono un repositorio?

    git clone .

  5. ¿Cómo puedo bajar los ultimos cambios de un repositorio?

    git pull (descarga y mezcla) o git fetch (solo descarga)

  6. ¿Cómo puedo gener un tag?

    Con el comando git tag -a v1.0 -m "mensaje".

  7. Cómo hago una mezcla de ramas?

    Sitúate en la rama receptora con git checkout main y luego ejecuta git merge nombre-rama.

  8. ¿Cómo puedo moverme a un commit especifico?

    Usando git checkout .

  9. ¿Cómo puedo borrar una rama?

    En local con git tag -d nombre-tag y en remoto con git push origin --delete nombre-tag.

  10. ¿Cómo puedo borrar un tag?

    git tag -d (local) o git push origin --delete (remoto).

  11. ¿Cómo puedo ver los cambios que estan por enviarse al repositorio?

    Con git status para ver archivos modificados y git diff --cached para ver las líneas exactas en el área de preparación.

  12. ¿Para que sirve el comando git remote -v?

    Para listar las URLs de los servidores remotos conectados al repositorio local.

  13. ¿Cómo le digo a git cual es mi email y correo?

    Mediante git config --global user.name "Tu Nombre" y git config --global user.email "tu@email.com".

  14. ¿Cuales son los pasos para conectar git con github?

    Crear repositorio en GitHub. git init en local. git remote add origin . git push -u origin master.

  15. ¿Cómo puedo crear un alias en git?

    Con el comando git config --global alias.nombre-corto comando-original.
